# Examples

Binary sample manuscripts are not committed. Use your own `.docx`, or create a small test document with text like:

```text
Ferroptosis was linked to cardiovascular injury (PMID: 35805165, 28776083).
Single-cell association proves causal remodeling. PMID: 29449364 12345678 23456789 34567890
```

Then run:

```powershell
biomed-ref run --docx sample.docx --out out
```
