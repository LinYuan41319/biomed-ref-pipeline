# Changelog

## 0.2.1

- Added CI, tests, documentation, and open-source readiness materials.

- Initial CLI workflow for PMID extraction, PubMed metadata fetch, RIS export, Zotero SQLite checks, JCR screening, citation planning, and manuscript QC.
